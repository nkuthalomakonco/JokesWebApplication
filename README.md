# Jokes Web Application

A web application built using **ASP.NET MVC** following the **Model-View-Controller (MVC)** pattern.

## Features

- Use the MVC (Model-View-Controller) design pattern.
- Configure database tables using the **Entity Framework**.
- Set up classes as **models** and customize **Razor forms**.
- Create **controller methods** to handle application logic.
- Style ASP.NET pages with **CSS**.
- Use **Bootstrap classes** for responsive and modern UI.
- Create a **search function** to filter database results.
